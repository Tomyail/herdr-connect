# 行为准则

[English](../../CODE_OF_CONDUCT.md)

## 我们的承诺

我们承诺让所有人都能在不受骚扰的环境中参与 Herdr Connect，不因年龄、体型、残障、族裔、生理性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教、性身份或性取向而受到区别对待。

我们会以有助于建立开放、友好、多元、包容和健康社区的方式行动与交流。

## 期望的行为

有助于建设正向社区的行为包括：

- 对不同观点和经历保持同理心与尊重；
- 提供并接受建设性反馈；
- 对错误承担责任、道歉并从中学习；
- 以项目和社区的整体利益为先。

不可接受的行为包括骚扰、带有性意味的言论或关注、挑衅或侮辱性评论、人身或政治攻击、蓄意恐吓，以及未经明确许可发布他人的隐私信息。

## 适用范围

本准则适用于项目空间，也适用于个人在公共空间正式代表项目时。项目空间包括仓库、Issue tracker、Pull Request、评审和其他官方沟通渠道。

## 报告与执行

请通过仓库的[私密漏洞报告页面](https://github.com/Tomyail/herdr-connect/security/advisories/new)私密报告行为问题，并注明这是社区行为报告。如果该私密表单不可用，请只创建一个请求维护者提供私密联系方式的公开 Issue，不要写入敏感事件细节。

维护者会及时审查报告，尊重报告者的隐私与安全，并采取公平、适度的措施。可能的处置包括要求纠正、警告、暂时限制或永久移出项目空间。不遵守或不执行本准则的维护者可能被移出项目团队。

## 来源

本准则改编自 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html)。
