# Triage 标签

工程技能使用以下五个固定角色；GitHub 标签字符串与角色名一致。

| 技能角色 | GitHub 标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 维护者尚未评估 |
| `needs-info` | `needs-info` | 等待补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，可交给 agent 继续拆票或实施 |
| `ready-for-human` | `ready-for-human` | 需要人工处理 |
| `wontfix` | `wontfix` | 明确不处理 |

修改右侧 GitHub 标签前，必须同步更新 tracker 中的实际标签，避免技能创建重复词汇。
