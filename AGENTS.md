# Herdr Connect 协作约定

## 文档语言

- 本项目所有面向人类阅读的文档统一使用简体中文，包括产品定义、领域语言、ADR、spec、计划、说明和交接文档。
- 为保证协议和代码的一致性，代码标识符、协议字段、枚举值、命令、文件名以及没有准确中文译名的专有名词可以保留英文，但其含义和取舍必须用中文说明。
- 引用外部资料时可以保留原文标题；项目自身的总结、结论和评审内容必须使用中文。

## Agent skills

### Issue tracker

项目使用 `github.com/Tomyail/herdr-connect` 的 GitHub Issues，通过 `gh` CLI 读写。详见 `docs/agents/issue-tracker.md`。

### Triage labels

项目使用五个默认 triage 角色标签。详见 `docs/agents/triage-labels.md`。

### Domain docs

项目采用 single-context 领域文档布局，统一领域语言记录在根目录 `CONTEXT.md`。详见 `docs/agents/domain.md`。
